Syllabus Overview

♣ Computer Graphics and Multimedia Application

UNIT-I

Introduction: The Advantages of Interactive Graphics, Representative Uses of Computer Graphics, Classification of Application Development of Hardware and software for Computer Graphics, Conceptual Framework for Interactive Graphics, Overview, Scan: Converting Lines, Scan Converting Circles, Scan Converting Ellipses.

UNIT-II

Hardcopy Technologies, Display Technologies, Raster-Scan Display System, Video Controller, Random-Scan Display processor, Input Devices for Operator Interaction, Image Scanners and Working exposure on graphics tools like Dream Weaver, 3D Effects etc, Clipping: Southland- Cohen Algorithm, Cyrus-Beck Algorithm, Midpoint Subdivision Algorithm.

UNIT-III

Geometrical Transformation: 2D Transformation, Homogeneous Coordinates and Matrix Representation of 2D Transformations, composition of 2D Transformations, the Window-to-Viewport Transformations, Introduction to 3D Transformations Matrix.

UNIT-IV

3D Graphics: Overview of 3D Graphics, 3D Transformations, Projection of 3D Objects, Introduction to the z-buffering, Ray tracing methods, and Shadow Generation.

UNIT-V

Multimedia Systems: Introduction to Multimedia, Components of Multimedia, Multimedia Applications in Education, Entertainment, Business, and Healthcare. Basics of Video and Audio compression techniques, Working with Multimedia Editing Tools.

UNIT-VI

Virtual Reality: Introduction to Virtual Reality, Hardware requirements for VR, Software for VR, Applications of Virtual Reality in Gaming, Education, and Training.

♣ Operating Systems

UNIT-I

Operating System Overview: Definition, Types, Functions, System calls, and Operating System Structures. Comparison of various operating systems and architectures.

UNIT-II

Process Management: Concept of a process, Process control block, Process states, Process scheduling, Scheduling algorithms (FCFS, SJF, RR, Priority), Inter-process communication.

UNIT-III

Memory Management: Memory Hierarchy, Memory allocation techniques (contiguous, paging, segmentation), Virtual memory, Page replacement algorithms (FIFO, LRU, Optimal).

UNIT-IV

File System Management: File concepts, File organization and access methods, Directory structure, File system implementation, and file system consistency.

UNIT-V

Security and Protection: Concepts of security, Types of threats, Security mechanisms (authentication, access control), Protection techniques.

UNIT-VI

Distributed Systems: Characteristics of distributed systems, Architectures, Communication in distributed systems, Challenges in designing distributed systems.

♣ Software Engineering

UNIT-I

Introduction to Software Engineering: Definition and importance, Software development life cycle (SDLC), Different software models (Waterfall, Agile, Spiral).

UNIT-II

Requirements Engineering: Requirement gathering and analysis, Requirement specification, Requirement validation.

UNIT-III

Design Concepts: Principles of software design, Design methodologies, Design patterns.

UNIT-IV

Software Testing: Testing fundamentals, Types of testing (unit testing, integration testing, system testing), Testing strategies.

UNIT-V

Software Maintenance: Types of maintenance, Maintenance processes, Software configuration management.

UNIT-VI

Software Project Management: Project planning, Scheduling, Risk management, Estimation techniques.

♣ Optimization Techniques

UNIT-I

Introduction to Operations Research: Definition, Importance, Scope, Applications of Operations Research in various fields.

UNIT-II

Linear Programming: Formulation of linear programming problems, Graphical method, Simplex method.

UNIT-III

Transportation and Assignment Problems: Formulation and solution of transportation problems, Assignment problems.

UNIT-IV

Dynamic Programming: Concept, Characteristics, Applications of dynamic programming.

UNIT-V

Game Theory: Introduction, Types of games, Solution methods for games.

UNIT-VI

Queueing Theory: Concepts, Basic models, Applications of queueing theory.

UNIT-I

Matrix Algebra: Introduction to matrices, Operations, Determinants, Inverse, Eigenvalues, and Eigenvectors.

UNIT-II

Differential Equations: First-order differential equations, Higher-order differential equations, Applications of differential equations.

UNIT-VI
